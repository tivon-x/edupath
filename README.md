# EduPath - 学习规划平台

EduPath 是一个帮助学生规划学业和职业发展的在线平台。

## 项目简介

这个平台旨在为学生提供：

- 个性化学习路径规划
- 职业发展指导
- 学习进度追踪

## 技术栈

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Shadcn UI

## 开发环境设置

1. 克隆项目

```bash
git clone [repository-url]
cd edupath
```

2. 安装依赖

```bash
pnpm install
```

3. 运行开发服务器

```bash
pnpm dev
```

访问 `http://localhost:3000` 查看项目。

## 项目结构

```
edupath/
├── app/          # Next.js 应用目录
├── components/   # React 组件
├── public/       # 静态资源
└── styles/       # 样式文件
```
